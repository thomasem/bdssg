# Static Site Generator

Based on BootDev's [Static Site Generator Project](https://www.boot.dev/learn/build-static-site-generator)

## Running site

```bash
./main.sh
```

Open [http://localhost:8888/](http://localhost:8888/) in your browser!

## Running tests

```bash
./test.sh
```
